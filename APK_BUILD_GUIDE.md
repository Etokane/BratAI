# Как собрать APK без Android Studio через GitHub

1. Зайди на github.com
2. Создай новый репозиторий, например `BratAI`
3. Загрузи туда все файлы из этой папки
4. Открой вкладку `Actions`
5. Нажми `Build APK`
6. Нажми `Run workflow`
7. Подожди 2–5 минут
8. Внизу появится файл `BratAI-debug-apk`
9. Скачай архив, внутри будет `app-debug.apk`
10. Перекинь APK на телефон и установи

Важно: Android может написать “установка из неизвестных источников запрещена”.
Нужно разрешить установку для браузера или файлового менеджера.
