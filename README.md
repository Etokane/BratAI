# Брат ИИ — личное Android-приложение

Это первая версия личного приложения для Николая.

## Что внутри

- Чат с ИИ
- Голосовой ввод
- Голосовой ответ
- Тёмная тема
- Персона: тёплый братский помощник
- Помощь с TikTok, хоррор-роликами, переписками и проверкой скама

## Как открыть

1. Установи Android Studio.
2. Нажми File → Open.
3. Выбери папку `BratAI_Android_Project`.
4. Дождись синхронизации Gradle.
5. Подключи телефон по USB или выбери эмулятор.
6. Нажми Run ▶.
7. В приложении вставь свой OpenAI API key и нажми OK.

## Важно

Это онлайн-версия. Ей нужен интернет и OpenAI API key.
Офлайн-версию можно сделать потом отдельно, но она будет слабее и тяжелее из-за локальной модели.

## Где менять характер ИИ

Файл:

`app/src/main/java/com/nikolai/bratai/MainActivity.kt`

Найди переменную:

`systemPersonality`

И меняй текст характера как хочешь.

## Если модель недоступна

В файле `MainActivity.kt` найди:

`"model", "gpt-5.5"`

И замени на модель, которая доступна в твоём OpenAI API-аккаунте.
